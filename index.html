Summary of Relationship Between Healthcare and Poverty

The data for this project is from the 2014 ACS 1-year estimates from the US Census Bureau. The above bubble chart shows percentage of those without heathcare compared to the percentage of those below the poverty line. Each bubble represents a different state.

There is a clear positive correlation between lack of healthcatre and poverty, where as the percent of the population of the state below the poverty line increases, the percentage of those lacking healthcare also increases.

The state with the highest percentage for lack of healthcare is Texas with 24.9%. The state with the lowest percentage lacking healthcare is Maine with 4.6%. For poverty percentages, the state with the highest is Mississippi with 21.5% and the lowest is New Hampshire with 9.2%.
